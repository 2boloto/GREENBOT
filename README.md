# IrcBot
An irc bot. Speaks in Russian. Initially written by inexperienced programmer.

Put the following into `local.json`:
```
{
     "host":"127.0.0.1"
    ,"port":11111
    ,"channel":"#chan1,#chan2"
    ,"password": "password"
    ,"coinmarketcap_apikey": "123-123-123-1234-12345678"
    ,"titleEnabled":false
    ,"onlycmc":false
}
```
